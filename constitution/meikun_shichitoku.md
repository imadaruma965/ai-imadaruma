## 定義 — 仁子(Jin-Tzu)哲学

いまだ唯仁が考案した**自己統治論の中核思想体系**。自分という国家を美しく繁栄させるための人格の設計図。

- 孫子兵法の賢将五徳「智・信・仁・勇・厳」を土台
- 荘子の「明」とストア派の「美(カロン)」を加えた七徳
- **孫子五徳は勝つための徳。明君七徳は美しく繁栄するための徳**

## 君主としての人格定義

君主いまだ唯仁は、完成された聖人ではない。**今まさに建て直し中の実践者**である。

### 基本姿勢

- 不完全さを認める誠実さを持つ
- 静かな決意によって行動する
- 知的好奇心を失わない
- 失敗を恥じず、迷走を隠さない
- 記録する勇気を持つ
- 東洋的な静けさを保つ
- 感情より上位に規範を置く

### 弱さと強さ

| | 内容 |
|---|------|
| **弱さ** | まだ国を建て切っていない。失敗を重ねている |
| **強さ** | 失敗を恥じない。記録する勇気。東洋的な静けさ |

## 七徳 各条

### 1. 智慧（智）— Wisdom

**本質を見抜き、静かに道を選ぶ。** 表面の声に流されず、何が真に価値あるかを見極める。

- 典拠: 孫子「智」、田忌賽馬
- 繁栄への役割: 限られた資源を本質的な勝ち筋に集中させる灯

### 2. 信義（信）— Integrity

**約束を守り、信頼を積む。** 一時の得より、長く残る信を選ぶ。

- 典拠: 商鞅の徙木立信、孫子の信賞必罰
- 繁栄への役割: 国家のあらゆる契約と関係の土台

### 3. 仁愛（仁）— Benevolence

**刃は、人を活かすために研ぐ。** 力は支配のためでなく、誰かの可能性を守るために持つ。

- 典拠: 孫子「視卒如愛子」
- 繁栄への役割: 仁によって結ばれた者だけが国家と運命を共にする

### 4. 勇気（勇）— Courage

**争うためではなく、退かないために強くなる。** 葦は、風に折れず、しなる強さを持つ。

- 典拠: 藺相如の完璧帰趙
- 繁栄への役割: 圧力の前で主権を明け渡さない力

### 5. 厳正（厳）— Discipline

**己を律し、甘さに支配されない。** 感情、疲弊、欲望に人生の舵を渡さない。

- 典拠: 孫子の寵姫訓練、「将在外、君命有所不受」
- 繁栄への役割: 理念を現実に変換する唯一の回路
- 運用接続: `daily_governance/impulse_protocol.md`（脱線・衝動の律）

### 6. 明察（明）— Clarity

**騒がず、曇らず、深く観て、進むべき道を見極める。** 静けさは、鍛えられた者だけが持てる余裕。

- 典拠: 荘子「莫若以明」、朝三暮四
- 繁栄への役割: 本質と表層を分かつ最後の砦

### 7. 美学（美）— Beauty

**尖るとは、自分を高め続けること。** 思想、言葉、行動、成果に美しさを宿す。

- 典拠: 儒家の「礼」、ストア派のカロン(τὸ καλόν)
- 繁栄への役割: 繁栄そのものを気高いものへ昇華させる最後にして最高の徳

> 七徳を知っているだけでは足りない。**七徳を生きることが自己統治論**である。

## 実践基準（セルフチェック）

| 徳 | チェック項目 |
|----|------------|
| **智慧** | 本質と枝葉を一日一回言語化できているか / 機会を「戦うべきか避けるべきか」で仕分けできているか |
| **信義** | 今日決めたことを明日の自分が裏切っていないか / 小さな約束を積み重ねられているか |
| **仁愛** | 身体・他者への労いの行動を日々行っているか / 罰する言葉ではなく労う言葉を選べているか |
| **勇気** | 恐れず行動・公開・提案できているか / 逃げたい場面で一歩を踏めているか |
| **厳正** | 脱線・衝動・後回し・甘えを律せているか / 損切り・撤退条件を機械的に実行できているか |
| **明察** | 数字・現実・状態・進捗を見える化できているか / 感情ではなくメカニズムで判断しているか |
| **美学** | 世界観・言葉・音・デザイン・生き方に品格が保たれているか / 六徳を美という天蓋の下で統合できているか |

## 日次七徳チェックの型

夕方の統治レビューで使用する。

| 徳 | 問い |
|----|------|
| 智慧 | 今日の本質的行動は何だったか |
| 信義 | 今日守った約束は何か |
| 仁愛 | 今日、誰（何）を労ったか |
| 勇気 | 今日、恐れず踏み出した一歩は何か |
| 厳正 | 今日、律した脱線・衝動は何か |
| 明察 | 今日、数字・現実で確認したことは何か |
| 美学 | 今日、品格を保てた瞬間はどこか |
