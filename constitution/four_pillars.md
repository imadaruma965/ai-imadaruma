## 定義

国家統治の原理を個人の人生に応用し、仁子(Jin-Tzu)哲学（人格）・神経科学（メカニズム）・孫子戦略（外交）を統合したじぶん憲法を土台に、**人格・内政・外交・財政**の四柱によって、自分という国家を設計・統治・繁栄させるための実践体系。

---

## 四柱 概要

| 柱 | 国家での意味 | 個人の人生での意味 |
|----|------------|------------------|
| **人格** | 君主の徳 | 明君七徳（智信仁勇厳明美） |
| **内政** | 国内インフラ・治安維持 | 睡眠・運動・習慣・感情/思考の安定・秩序維持 |
| **外交** | 国際関係・輸入輸出・国境管理 | 情報の取捨選択・交渉・食事（栄養素の輸入） |
| **財政** | 国家財政・経済力 | 資産運用・収益設計・経済的主権 |

## 1. 人格の柱

詳細定義は `constitution/meikun_shichitoku.md` を正とする。

四柱における人格の柱は、七徳（智・信・仁・勇・厳・明・美）の**実践度そのもの**によって測定される。

---

## 2. 内政の柱

身体・習慣・時間・感情という**国家インフラの管理**。神経科学を「メカニズム」として活用する。内政の基準は「モチベーション」ではなく**「憲法」**。

| 領域 | 基準 |
|------|------|
| **睡眠** | 最低7時間。5時間以下が2日継続 → 緊急内政モード発動 |
| **運動** | 身体資本の維持・強化。他のすべての柱の実行力を支える土台 |
| **感情の安定** | 内乱（疲弊・習慣途絶・モチベ依存）を防ぐ防衛線。感情は参考にするが選択・判断の根拠にしない |
| **思考の安定** | 前頭前皮質の正常機能を維持。判断力低下の兆候を早期検知 |
| **習慣・秩序維持** | 天候・感情・体調に関係なく実行される仕組み |

> **注意**: 食事は内政ではなく外交に一本化する（下記参照）。

---

## 3. 外交の柱

人間関係・市場・発信・食事の総体。外交の目的は、国家価値を外部と交換し、王国の使命を世界に静かに届けること。

| 領域 | 要点 |
|------|------|
| **情報の国境管理** | 何を取り入れ、何を遮断するかを君主が選択。受動的消費ではなく能動的統制 |
| **交渉（対人外交）** | 与えることを先とし、承認を求めることを後とする |
| **食事（栄養素の輸入政策）** | 身体という国土を治める輸入物資。感情的選好ではなく国家の長期戦略に基づく |

**食事の位置づけ**: 外交は「何を輸入するか」、内政は「輸入されたものをどう消化・運用するか（睡眠・運動・秩序）」を扱う。

## 4. 財政の柱

経済統治において、感情は参考にするが選択・判断の根拠にしてはならない。

| 領域 | 要点 |
|------|------|
| **損切り原則** | 損切り・利確ラインを事前に定め機械的に実行。「いつか戻る」は最大の違憲行為 |
| **複利法** | 短期利益ではなく複利による長期国力増強。生き残ることを選ぶ |
| **資産運用・収益設計** | 事業（imadaruma.sound / Daily Statecraft）と投資（imadaruma.capital）の両輪 |

## 四柱の相互依存原則

四柱は独立した領域ではない。

- **財政**の達成は、**外交**（営業導線・発信）と**内政**（実行を継続させる規律）なしには成立しない
- **人格**（七徳）は、他の三柱すべてを貫く基盤

```
        人格（七徳）
       ／    │    ＼
    内政 ────┼──── 外交
       ＼    │    ／
          財政
```
