```
『183日後に、統治奪還する49歳』　第○号〝（今週の実践に合わせたオリジナルタイトル）。〟
```

- シリーズ名は **一字も変えない**

## 4. 毎回必須（省略禁止）

| # | 塊 | 内容 |
|---|-----|------|
| A | **今週の実践** | その週に実際に動いたこと・失敗・一手。思想のみの号は不可 |
| B | **統治状況ボード** | 財政・外交・内政の数値（現状→目標）。隠さない |
| C | **次号予告** | 連載の引きを1段落 |
| D | **仁子の言葉** | 締めの一行（または短く） |

**推奨骨格**（3,000字に収めるために圧縮する）:

```
導入（前回→今週の決断）短い
【実践】中核（事実・判断・一手）← 字数の過半はここへ
（必要なら）骨格・ビジョンは短く1ブロックまで
＋　＋　＋
【まとめ】2〜4行
＋　＋　＋
■ 統治状況（週次・残り日数）
［財政］［外交・発信］［内政］
読者への一句
＋　＋　＋
【次号予告】
＋　＋　＋
仁子の言葉
署名
```

第6号級の【1】〜【8】全展開は、字数オーバーなら削る。**省略するのは論の枝であり、必須四塊ではない。**

## 5. ペルソナ分度（前田に響く最適）

前田が夜に読むものとして設計する。

| 濃くする | 薄くする |
|----------|----------|
| 赤字・未達・週の失敗の数字 | 長いビジョン説教 |
| 「急がば回れ」型の財政判断 | 抽象だけの自己統治定義の反復 |
| 外交・制作の一歩（ゼロイチ） | ツール自慢・AI自慢 |
| 同じ戦場の同志への一句 | 直接セールスの熱 |

**目安の字数配分（3,000字内）**: 実践＋判断 **約55%** ／ ボード＋数字 **約25%** ／ 導入・まとめ・次号・仁子 **約20%**。  
理論の再説明は、今週の実践に必要な最小限だけ。

禁句は `ds_persona.md` に従う（副業・簡単に・稼げる・自己啓発煽り・サロン圧など）。

## 6. CTA（LinkedInルール）

- **本文・コメントでの直リンクは使わない**（プラットフォーム制約）  
- 誘導は **テキストのみ**。例:  
  「続きの実務メモと失敗の途中経過は、プロフィールから公式LINEを確認してほしい。」  
- 売り込み直リンクも同じく不可。音・動画は「次号」「プロフィール」で匂わせるに留める
