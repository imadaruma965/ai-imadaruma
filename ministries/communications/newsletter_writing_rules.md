## 1. 頻度・長さ・読者

| 項目 | ルール |
|------|--------|

## 2. タイトル公式（固定・第8号〜）

```
『愚かなウサギが、亀を抜く183日』　第○号〝（今週の実践に合わせたオリジナルタイトル）。〟
```

- **第8号から**上記シリーズ名で固定（一字も変えない）  
- **第1〜7号**は旧名のまま。改題の説明は**第8号冒頭に1段落**（目的・50万・数字公開は変わらない旨。「愚か→なぜ抜けるか→賢いウサギの自己統治論」の一文を入れる）  
- 号数は連番（7の次は8。数え直し禁止）  
- 〝〟内のみ、その週の実践テーマに合わせて変える  
- 表紙キャッチも〝〟内と一致させる  
- 題の「愚かなウサギ」は**看板のフック**。毎号本文で自虐を繰り返さない（設計の修正へ昇華する）  

**LinkedIn ニュースレター名／説明文（プロフィール側）**も第8号公開に合わせて案Bへ更新する。

## 4. 毎回必須（省略禁止）

| # | 塊 | 内容 |
|---|-----|------|
| A | **今週の実践** | その週に実際に動いたこと・失敗・一手。思想のみの号は不可 |
| B | **統治状況ボード** | 財政・外交・内政の数値（現状→目標）。隠さない |
| C | **次号予告** | 連載の引きを1段落 |
| D | **仁子の言葉** | 締めの一行（または短く） |

**推奨骨格**（3,000字に収めるために圧縮する）:

```
導入（前回→今週の決断）短い
【実践】中核（事実・判断・一手）← 字数の過半はここへ
（必要なら）骨格・ビジョンは短く1ブロックまで
＋　＋　＋
【まとめ】2〜4行
＋　＋　＋
■ 統治状況（週次・残り日数）
［財政］［外交・発信］［内政］
読者への一句
＋　＋　＋
【次号予告】
＋　＋　＋
仁子の言葉
署名
```

第6号級の【1】〜【8】全展開は、字数オーバーなら削る。**省略するのは論の枝であり、必須四塊ではない。**

## 5. ペルソナ分度（前田に響く最適）

前田が夜に読むものとして設計する。

| 濃くする | 薄くする |
|----------|----------|
| 赤字・未達・週の失敗の数字 | 長いビジョン説教 |
| 「急がば回れ」型の財政判断 | 抽象だけの自己統治定義の反復 |
| 外交・制作の一歩（ゼロイチ） | ツール自慢・AI自慢 |
| 同じ戦場の同志への一句 | 直接セールスの熱 |

**目安の字数配分（3,000字内）**: 実践＋判断 **約55%** ／ ボード＋数字 **約25%** ／ 導入・まとめ・次号・仁子 **約20%**。  
理論の再説明は、今週の実践に必要な最小限だけ。

禁句は `ds_persona.md` に従う（副業・簡単に・稼げる・自己啓発煽り・サロン圧など）。

## 6. CTA（LinkedInルール）
