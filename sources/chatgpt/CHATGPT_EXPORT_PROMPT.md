# ChatGPT用・書き出しプロンプト（コピペ）

壁打ちの**最後**に、そのまま送る。

---

以下を、このスレの内容だけから書いてください。創作しない。分からない項目は「未出」と書く。

1. **題名**（10〜20字）
2. **領域タグ**（該当のみ）: sound / DailyStatecraft / Instagram / 料金 / その他
3. **今日決まったこと**（箇条書き・最大7）
4. **まだ決めていないこと**（箇条書き）
5. **捨てた案**（あれば）
6. **数字・価格・期限**（出てきたものだけ）
7. **次にやる一手**（最大3・動詞で）
8. **全文ログ**（このスレのユーザー発話とあなたの発話を時系列で残す。要約で消さない）

出力は Markdown のみ。見出しは上記番号に合わせる。
