# parking_lot — 保留・後回し置き場

> 今はやらないが、消さないアイデア・タスク・衝動の記録場所。
> 統治を止めないために、ここに「停める」。

## 運用ルール

- 脱線したアイデア、衝動の内容、今は不要なタスクをここに置く
- **判断は後でする**。置くときは1行で十分
- 週次レビューで整理する（採用 / 却下 / 先送り）
- 採用したものは翌週の閣議で統治計画に組み込む
- 却下したものは削除してよい

## 追加テンプレート

新しいアイテムを追加するときは、以下をコピーして使う。

```
| YYYY-MM-DD | （1行で内容） | （衝動/閣議/その他） | ⬜保留 |
```

---

## 週次整理ルール

毎週日曜の週次レビューで実施する。

1. 保留中のアイテムをすべて確認する
2. 各アイテムに判定をつける:
   - **採用** → 翌週の閣議で統治計画に組み込む
   - **却下** → 削除する（理由を1行メモしてよい）
   - **先送り** → そのまま保留
3. 2週間以上保留のままのアイテムは、採用か却下を決める
