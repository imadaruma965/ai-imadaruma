# parking_lot — 保留・後回し置き場

> 今やらないことを一時保管し、**今日の最重要任務を守る**ための兵站。
> 脱線を責めず、記録し、後で判断する。

---

## 目的

- 思いつき・脱線・依頼・調査欲を**今すぐ処理せず**ここに置く
- 今日の主戦場と最低売上行動を守る
- 判断は後回しにする。置くときは1行で十分
- 週末の `weekly_review.md` で棚卸しする

---

## AI尊徳の役割

| 項目 | 内容 |
|------|------|
| **立場** | 兵站の管理者。脱線を責めない |
| **する** | 記録を促す、週末整理を案内する、採用時は `morning_cabinet.md` へ接続する |
| **しない** | 即座に戦略を変更しない（戦略判断はAI栄一へ） |
| **補助原理** | 蕭何 — 兵站・記録・継続運用 |

---

## `impulse_protocol.md` との関係

```
脱線・衝動が起きる
  ↓
impulse_protocol.md で3分復帰（最優先）
  ↓
復帰後、今日やらない内容は parking_lot.md に1行で記録
  ↓
主戦場に戻る
```

- **impulse_protocol** = 今すぐ戻る（復帰）
- **parking_lot** = 今はやらないが消さない（保管）

| 発生日 | 思いつき・脱線内容 | 分類 | 今やらない理由 | 次に確認する日 | AI栄一に判断を求めるか | 判定 |
|--------|------------------|------|--------------|--------------|---------------------|------|
| YYYY-MM-DD | | | | YYYY-MM-DD | ⬜はい / ⬜いいえ | ⬜採用 / ⬜保留 / ⬜廃棄 |

### 分類

| 分類 | 例 |
|------|-----|
| **事業** | 新商品アイデア、価格変更、顧客対応 |
| **発信** | SNS投稿案、記事ネタ、LinkedIn |
| **制作** | デザイン、資料、コンテンツ制作 |
| **投資** | 銘柄調査、ポートフォリオ変更 |
| **学習** | 読書、講座、スキル習得 |
| **個人** | 私事、家族、健康 |
| **その他** | 上記に当てはまらないもの |

### 判定

| 判定 | 意味 | 次のアクション |
|------|------|--------------|
| **採用** | 来週以降の統治計画に組み込む | `morning_cabinet.md` でタスク化 |
| **保留** | まだ判断しない | 次回確認日まで待つ |
| **廃棄** | やらないと決める | 行を削除（理由を1行メモ可） |

### AI栄一に判断を求める場合

以下に該当するときは「はい」を選び、週末 `weekly_review.md` でAI栄一へ相談する。

- 新しい収益源・事業の追加
- ロードマップの「やらない戦」と矛盾しそう
- 採用すると主戦場が変わる可能性がある

## 週末棚卸し（`weekly_review.md`）

毎週末、`weekly_review.md` のセクション7で実施する。

1. 保留中のアイテムをすべて確認する
2. 各アイテムに判定をつける（採用 / 保留 / 廃棄）
3. **採用** → 来週の `morning_cabinet.md` で統治計画に組み込む
4. **廃棄** → 行を削除する
5. **保留** → 次に確認する日を更新する
6. 「AI栄一に判断を求める」が ⬜はい の項目は、週末レビューでAI栄一へエスカレーション
7. 2週間以上保留のままのアイテムは、採用か廃棄を決める
