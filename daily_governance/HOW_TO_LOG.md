# 数字・実践の打込み手順（これだけやれ）

> 迷ったらこのファイルだけ見る。  
> 日報を癖にする段階設計 → `daily_journal_roadmap.md`  
> 方針の長い話 → `strategy/metrics_capture_note.md`

---

## 道具は2つだけ（覚え方）

| いつ | どこに打つ | 所要 | 忘れると |
|------|------------|------|----------|
| **毎晩（主）** | `night_review.md` の **§8b** | 2〜5分 | 習慣が死ぬ |
| **金曜〜土曜** | `governance_board.md` | 10〜15分 | 日報があれば後から拾える |

週次ボードより **毎晩の日報を先に癖にする**（理由と層は `daily_journal_roadmap.md`）。


チャットにメモして終わりにしない。必ず上記ファイルに書く。  
打った夜レビューは、いつもの通り `archives/daily_logs/YYYY-MM-DD.md` にコピーする。

---

## ① 毎晩（実践）— やり方

1. Cursor（またはエディタ）で `daily_governance/night_review.md` を開く  
2. 日付を今日にする  
3. 上から普通に夜レビューする（短くてよい）  
4. **§8b「今日の実践ログ」だけは必ず埋める**  

§8bの最低ライン（これで合格）:

```
今日動いた事実: （例）コーチにLinkedIn申請5件／sound草案1／筋トレした
失敗・揺れ:     （なければ「なし」）
```

数字欄は **空でよい**（毎日フォロワーを書かない）。

5. ファイル全体を `archives/daily_logs/2026-07-14.md` のようにコピー保存  
6. 終わり

スマホだけでやりたい夜は、メモアプリに3行→翌朝 §8bへ転記でも可。ただし翌朝うちにファイルへ。

---

## ② 金曜〜土曜（数字＋実践の確定）— やり方

1. `daily_governance/governance_board.md` を開く  
2. **週のメタ**（週番号・残り日数・NL号数）を埋める  
3. エクスプローラで `archives/daily_logs/` の **直近7日** を開き、§8bを眺める  
4. **A欄**に、ネタになりそうな実践をまとめる（動いた事実／失敗／一手／次号の種）  
5. **数字**を打つ——各アプリを開いて目視し、ボードの表に転記する（週1回だけでよい）

| 見る場所の例 | ボードの欄 |
|--------------|------------|
| 銀行・家計簿・請求 | B. 財政 |
| LinkedIn／NL管理画面 | C. 外交・発信 |
| LINE公式管理 | C. LINE |
| Instagram／YouTube | C. 同左 |
| 体重計 | D. 内政 |

測っていない欄は **「未計測」** と書く。空欄で悩まない。

6. 「伏せたいもの」に印  
7. チェックリスト（E）を付ける  
8. 正篤に渡す：「`governance_board.md` を読んでNLを書け」

日曜NL公開後: ボードを `archives/governance_boards/YYYY-Www.md` にコピー → ボードを翌週用に空ける。

## ③ 打たなくてよいもの

- 同一内容の Notion／スプレッドシート二重入力（段階0では禁止）  
- チャットだけの日報（ファイルに残らないと正篤が使えない）
