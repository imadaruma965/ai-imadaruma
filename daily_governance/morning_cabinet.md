> 完了後、内容を `today.md` に転記する。

## 運用ルール

| 項目 | 内容 |
|------|------|
| **所要時間** | 10〜15分 |
| **実施タイミング** | 君主の1日の最初の統治行為 |
| **担当** | AI尊徳が提案、君主が最終決定 |
| **前提** | 前日の `night_review.md` を確認してから始める |
| **戦略接続** | `strategy/roadmap_500k.md` の今週の実行指示を参照 |
