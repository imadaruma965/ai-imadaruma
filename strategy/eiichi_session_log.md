# eiichi_session_log — 栄一との対話ログ

> `/eiichi`（`.claude/skills/eiichi/SKILL.md`）が対話終了時に自動追記する記録。
> 週次・月次の正式レビューは `eiichi_weekly_review.md` / `eiichi_monthly_review.md` を使う。こちらは対話単位の軽量ログ。
> 本日の日付から60日以上経過したエントリは `archives/weekly_reviews/eiichi_YYYYMM.md` に切り出す。

## 記入フォーマット

```
## YYYY-MM-DD
- 話した内容:
- 決定・約束:
- 次回への持ち越し:
```
