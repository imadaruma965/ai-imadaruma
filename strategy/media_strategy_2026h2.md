## 0. 栄一の判断

母体は二つ。曜日の柱は **四柱に固定** し、各SNSは同じ軸に合わせる。

1. **カルーセル週4**（火〜金）＝四柱そのもの  
2. **ニュースレター週1**（日）＝183日実録  

原則:

> **火〜金は「その日の柱のカルーセル1本」がすべてのSNSの母体。日曜だけNLが母体。**

## 1. 曜日の軸（全SNS共通）

| 曜日 | 柱 | 母体 |
|------|-----|------|
| **火** | **人格**（明君七徳に触れる） | カルーセル |
| **水** | **内政** | カルーセル |
| **木** | **外交** | カルーセル |
| **金** | **財政** | カルーセル |
| **日** | （実録横断） | **NL（聖域）** |
| **月** | （制作・分岐日） | 新規投稿義務なし。音源アップ・解説・予約確認 |
| **土** | （余力） | 任意。なければ休む |

**火曜・人格の中身**: 明君七徳（智・信・仁・勇・厳・明・美）のいずれかに触れる。正本は `constitution/meikun_shichitoku.md`。憲法条文の解説枠ではない（じぶん憲法は全投稿の上位法として残す）。

---

## 2. 火〜金の配信（同一ネタ・同日）

その日のカルーセル1本を母体に、次を **同じ柱・同じ日** で流す。

| 媒体 | 何をするか |
|------|------------|
| **LinkedIn** | カルーセル投稿（オリジナル母体） |
| **Facebook** | LIと **同じ投稿を共有** |
| **Instagram** | カルーセルを IG用に変換して配信 |
| **X** | 同カルーセルから短文（その日の柱） |

新規キャンペーンを媒体別に作らない。変換・短縮・比率合わせのみ。

---

## 3. 日曜 NL と分岐

| 成果物 | 媒体 | ルール |
|--------|------|--------|
| **NL本文** | LinkedIn | 約3,000字・必須四塊・直リンク禁止（執筆ルール） |
| **音声化** | note / Podcast / YouTube | **音源は1本**を配る |
| **解説動画** | YouTube | NLネタ。音読とは別だが台本はNL派生 |

推奨: 日＝NL公開＋収録／月＝各所アップ＋解説。取れなければ音声優先、解説は繰り越し可。

## 4. 配管図

```
【母体A】カルーセル ×4
  火=人格（明君七徳） / 水=内政 / 木=外交 / 金=財政
    → 同日: LinkedIn / Facebook共有 / Instagram変換 / X短文

【母体B】日曜 NL
    → LinkedIn
    → 音声1本 → note / Podcast / YouTube
    → 解説動画1本
```

## 5. 生産リズム

| いつ | 誰 | 何 |
|------|-----|-----|
| **金** | 正篤 | 翌週カ4（火人格〜金財政）＋日曜NL下書き（入力は `governance_board.md`） |
| **金〜土** | 君主 | **統治ボード15分**（実践・数字）。これが無いと正篤は書けない |
| **土** | 君主 | NL確認。LI/FB/IG/X を火〜金分予約 |
| **日** | 君主 | NL公開 → 音声収録。ボードを `archives/governance_boards/` へ |
| **月** | 君主 | 音源配布・解説・予約の最終確認 |

数値方針: `strategy/metrics_capture_note.md`（アプリ先行は否。段階0＝ボード1枚）

**分度**: 形変え実作業は週 **90〜120分**。超えたら解説→Xの文量から削る。  
**聖域**: 日曜NL。次点は火〜金カ（理想4・下限は柱を落とさず本数を減らすなら財政or外交から）。

優先順: **ボード記入 → NL → 火〜金カ → FB/IG同日 → 音声 → X → 解説**。
