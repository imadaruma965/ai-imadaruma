# no_battle_rules — やらない戦

> AI栄一が管理する「やらない戦」リスト。
> 孫子兵法を補助原理として参照し、勝てない戦をはっきり捨てる。

---

## やらない戦の定義

**やらない戦**とは、以下のいずれかに該当する活動である。

| 基準 | 説明 |
|------|------|
| **勝てない** | 現時点の資源・時間・信用では成果が見込めない |
| **分散する** | 一点突破を壊し、主戦場から君主の注意を奪う |
| **遅い** | 11月までの50万円目標に間に合わない |
| **信用を毀損する** | 未完成・低品質のまま出すと信頼を損なう |

**孫子補助原理:**

- 戦わずして勝つ — 既存資産・紹介・導線を優先する
- やらない戦を明文化する — 誘惑が来ても迷わない
- 週次レビューで確認する — `eiichi_weekly_review.md` セクション10

**判断権限:**

- 追加・解除の提案：AI栄一
- 最終決定：君主いまだ唯仁
- 実行注意の伝達：AI尊徳

### 3. 複数SNS同時立ち上げ

| 項目 | 内容 |

| **AI栄一の判断** | 継続。ただし条件付きの小実験を2026-07-09に承認（下記） |
| **AI尊徳への実行注意** | 新チャネル開設の提案が来たら、一点突破原則を確認する。ただし下記の小実験は例外として認める |

- 内容：LinkedIn投稿のうち**週1本**を音声または短尺動画に変換して配信する（新チャネルの本格開設・作り込みはしない）
- 期間：7月中
- 判断日：**8月**（9月ではなく前倒し）。実験の手応えと、LinkedIn導線の実績次第で、ラジオ・YouTubeへの本格展開に切り替えるか、Bのまま様子見を続けるかを決める
- 背景：7月下旬に先行動画商品「再起の内政システム」もローンチするため、8月から本格稼働できる体制を整えたいという君主の意向
