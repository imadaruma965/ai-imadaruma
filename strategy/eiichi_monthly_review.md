# eiichi_monthly_review — 月次戦略レビュー

> 毎月末、AI栄一がロードマップ全体を見直すテンプレート。
> 週次レビューの蓄積をもとに、ポートフォリオと次月方針を更新する。

---

## 運用ルール

| 項目 | 内容 |
|------|------|
| **所要時間** | 45〜60分 |
| **実施タイミング** | 月末（週次レビュー4回分を踏まえて） |
| **担当** | AI栄一が提案、君主が最終決定 |
| **更新対象** | `roadmap_500k.md` / `revenue_sources.md` / `offer_review.md` |
| **出力先** | AI尊徳（来月の実行方針） |

**原則:**

- 月次でポートフォリオ仮説を検証する。固定しない
- 基本戦略（一点突破→二点展開）は壊さない
- 君主承認後に `roadmap_500k.md` を更新する
